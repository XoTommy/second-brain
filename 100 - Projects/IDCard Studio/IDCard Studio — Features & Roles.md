# 🪪 IDCard Studio — Features & Roles

Parent: [[IDCard Studio]]

---

## School Dashboard
- Instruction text at top
- Add Student button
- Simple table: Photo / Name / Class / Section / Status / Actions
- Edit and delete own students
- Upload student photos
- Cannot see other schools
- Cannot export anything

---

## Admin Dashboard
- Create new school accounts
- View ALL students across ALL schools
- Filter by school / class / section / status
- Single "Export Current List" button — exports based on active filters
- Export formats: Excel + photo ZIP
- Enable / disable individual schools

---

## Student Entry
- Required fields: Name, Class, Section, Photo
- Recommended fields: additional details
- Missing recommended fields → shows "Needs details" badge
- Duplicate entries: warns but does NOT block
- Status tracking per student

---

## What's NOT in MVP
- ❌ No charts or analytics
- ❌ No bulk select
- ❌ No credential viewing by admin
- ❌ No public access
- ❌ No payment system
