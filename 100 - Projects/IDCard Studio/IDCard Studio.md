## Two Roles
### School
- Add / edit / delete students
- Upload student photos
- Cannot export

### Admin (Family)
- Create school accounts
- View + filter all students across all schools
- Export filtered Excel + photo ZIP
- Enable / disable schools

---

## Business Context
- Built for a family ID card printing business
- Schools submit student data → family prints ID cards
- Replaces manual/WhatsApp-based data collection
- Real orders are waiting — speed is top priority
