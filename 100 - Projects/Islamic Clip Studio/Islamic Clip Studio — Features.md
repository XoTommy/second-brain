# 🎬 Islamic Clip Studio — Features

Parent: [[Islamic Clip Studio]]

---

## Core Flow
1. User pastes YouTube URL or uploads audio/video file
2. yt-dlp downloads the content
3. AssemblyAI transcribes with Arabic + English accuracy
4. AI (Groq/llama) identifies best Islamic moments
5. FFmpeg cuts clips at identified timestamps
6. Subtitles burned into clips
7. User downloads ready-to-post clips

---

## AI Clip Intelligence
- Tuned specifically for Islamic content
- Identifies: akhirah moments, tawheed points, dawah hooks, emotional peaks
- Understands Islamic terminology — not generic speech detection
- Prioritizes moments that perform well as short form content
