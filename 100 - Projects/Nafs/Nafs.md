## 🧭 Core Philosophy
- Muhasabah — Islamic self-accountability, not gamification
- No streaks, no scoring UI, no pressure
- Premium feel — dark, calm, intentional
- Religious text is NEVER paraphrased — exact wording only
- Storage schema changes require explicit approval before touching
- Approve plan before any code — surgical patches only

---

## 👥 Target User
- Practising Muslims who take their deen seriously
- Age 18–35, globally distributed
- Frustrated by shallow "Islamic" apps that feel like generic habit trackers
- Wants honest self-reflection, not dopamine loops

## ⚙️ Dev Rules (Non-Negotiable)
1. Approve plan before any code
2. Surgical file-scoped patches only
3. Additive-only changes — zero regression risk
4. Explicit self-audit table + test checklist on every patch
5. Storage schema changes need explicit approval
6. Religious text never paraphrased
7. Uses NZT-mode for deep pre-implementation analysis
